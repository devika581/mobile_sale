Mobile Sales Analysis Dashboard
Project Overview
This project focuses on analyzing mobile sales data to understand sales performance, customer purchasing trends, and product demand. The dashboard provides insights that help businesses make data-driven decisions.
Tools & Technologies
Microsoft Excel
Power BI
Project Files
mobile_sales.xlsx – Dataset used for analysis
MOBILE_SALES.pbix – Interactive Power BI Dashboard
Project Objectives
Analyze overall mobile sales performance
Identify top-selling mobile brands and models
Track sales trends over time
Understand customer purchasing patterns
Monitor revenue and sales KPIs
Dashboard Features
Total Sales Overview
Brand-wise Sales Analysis
Model-wise Performance
Monthly Sales Trends
Revenue Analysis
Interactive Filters and Visualizations
Key Insights
Best-performing mobile brands
Top-selling mobile models
Sales trends across different periods
Revenue contribution by products
Outcome
This project demonstrates data cleaning, dashboard development, data visualization, and business intelligence skills using Excel and Power BI.
